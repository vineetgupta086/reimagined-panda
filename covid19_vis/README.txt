This is a data visualisation project for a couple weeks of beginning of COVID-19 spread.
The comparison is done on China's data with that of the rest of the world. 

Source of data:
https://github.com/RamiKrispin/coronavirus

Please note that information and data regarding COVID-19 is frequently being updated.
The data used in this project was pulled on March 17, 2020, and should not be considered
to be the most up to date data available.
