Project: COVID-19 Time Series Forecasting
The forecasting is done for number of new confirmed cases and new deaths on a daily basis in India.
First data till October end is obtained and predictions are made for the month of November and then the predictions
are also compared with the actual data recorded in November.

Source of data:
www.data.world/shad/covid-19-time-series-data
https://github.com/datasets/covid-19/blob/master/data/countries-aggregated.csv
