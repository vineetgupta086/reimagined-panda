Project: COVID-19 Time Series Forecasting
The forecasting is done for number of daily newly infected cases in India.
Data is scrapped off of another github link, till 31 December, 2020 and predictions have been made till 31 January, 2021.

Source of data:
covid19.who.int
https://github.com/datasets/covid-19/blob/master/data/countries-aggregated.csv
